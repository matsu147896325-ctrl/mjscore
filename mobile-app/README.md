# 麻雀スコア集計 スマホ版

このフォルダは、iPhone・Androidのブラウザで使えるスマホ版です。

## 外出時にも使う方法

外出時に使うには、PC内の `http://192.168...` ではなく、HTTPSで公開します。
公開後、スマホで一度開いてホーム画面に追加すると、アプリのように起動できます。

おすすめは以下のどちらかです。

## 方法1: GitHub Pages

1. GitHubでリポジトリを作成します。
2. この `mobile-app` フォルダの中身をリポジトリへアップロードします。
3. GitHubの `Settings > Pages` を開きます。
4. `Deploy from a branch` を選びます。
5. `main` ブランチの `/root` を公開します。
6. 表示された `https://...github.io/...` のURLをスマホで開きます。

## 方法2: Netlify

1. https://www.netlify.com/ を開きます。
2. `mobile-app` フォルダをドラッグ&ドロップで公開します。
3. 表示された `https://...netlify.app` のURLをスマホで開きます。

## ホーム画面に追加

iPhone:

1. Safariで公開URLを開きます。
2. 共有ボタンを押します。
3. `ホーム画面に追加` を選びます。

Android:

1. Chromeで公開URLを開きます。
2. メニューを開きます。
3. `ホーム画面に追加` または `アプリをインストール` を選びます。

## オフライン利用

公開URLを一度開いたあと、スマホに保存されます。
その後は通信が弱い場所でも起動しやすくなります。

ただし、初回だけはインターネット接続が必要です。
